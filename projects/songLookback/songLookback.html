Song Lookback is a web application that captures your portrait including your favorite song in a given month using p5.js and Spotify API.

Song Lookback was designed to use the Google Photos API to get photo metadata to send to Spotify API and play a user’s top track at the date photo was taken. Unfortunately, Spotify API only allows you to get user top tracks, compounded, for the past month, six months and all time. They have ‘recently played tracks’, where you can input a UNIX timecode, but only for the last 50 songs. Google Photos API only allowed you to get metadata after using a media ID parameter. I wanted to extract exif data, locally within a familiar app like Google Photos, Instagram or iPhotos.

Future iterations will include a toggleable disco mode, proper photo booth and scapbook gallery.
